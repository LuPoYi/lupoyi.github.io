GIT

apt-get install git-core

git config --global user.name "name"
git config --global user.email "email"

git config --list #查看設定

git status

git add
git add .

git commit
git commit -m "Add test.rb to test git function"

git log (-p)
git log --stat

gitk --all (&) 內建gui
git branch 這個指令可以列出所有的 branch 並告訴你目前正在哪個 branch：
git branch (new_branch_name)

git checkout "Loser" 切換branch

http://howar31.blogspot.tw/2013/06/git.html http://tech.marsw.tw/blog/2013/08/16/git-notes-github 因此我們稍微做一下流程的整理： 修改檔案 => 加入 stage (git add) => 提交( git commit )=> 繼續修改其他檔案
